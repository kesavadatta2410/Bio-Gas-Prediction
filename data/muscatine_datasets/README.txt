This is the repository for laboratory, operational, and SCADA datasets for anaerobic co-digestion at the Muscatine, IA Water Resource Recovery Facility from 2020-2023. 

The following datasets are available for download and use:
LABS-raw - a file with daily lab and operational data that has been minimally pre-processed, as described in the data dictionary
SCADA-raw - a file with raw, minute-scale SCADA information, with the following pre-processing: 485 rows assumed to be power surges were deleted, and variable names were changed to be interpretable
